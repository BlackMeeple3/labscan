# LAB·SCAN

App mobile per la gestione campioni da laboratorio.

## Tecnologie
- React + Vite
- Tesseract.js (OCR gratuito, gira nel browser)
- Web Speech API (riconoscimento vocale nativo)
- Zero costi, zero API key esterne

## Deploy su Vercel via GitHub

1. Carica questa cartella su un repository GitHub
2. Vai su vercel.com → New Project → importa il repo
3. Vercel rileva Vite automaticamente → clicca Deploy
4. In 2 minuti hai il link dell'app

## Sviluppo locale

```bash
npm install
npm run dev
```
